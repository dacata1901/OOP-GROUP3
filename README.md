# 🎮 Game Đoán Từ (Word Guessing Game) – OOP C++

## 📌 Giới thiệu
Đây là project Game Đoán Từ được xây dựng bằng ngôn ngữ C++ và áp dụng các nguyên lý Lập trình Hướng Đối Tượng (OOP).

Người chơi sẽ đoán từ tiếng Anh bị ẩn một phần ký tự tùy theo độ khó.

---

## 🚀 Chức năng chính
- Đọc dữ liệu từ file `.txt`
- Cho phép chọn độ khó:
  - Easy (dễ)
  - Medium (trung bình)
  - Hard (khó)
- Ẩn các ký tự ngẫu nhiên trong từ
- Cho phép người chơi nhập lại nếu sai
- Chuyển sang từ mới nếu đoán đúng
- Cho phép tiếp tục hoặc dừng game

---

## 🧠 Áp dụng OOP
Project sử dụng các khái niệm:

- **Encapsulation (Đóng gói)**  
  → Mỗi class có chức năng riêng biệt

- **Inheritance (Kế thừa)**  
  → Các class kế thừa từ `GameComponent`

- **Polymorphism (Đa hình)**  
  → Sử dụng hàm `virtual`

- **STL (Standard Template Library)**  
  → Sử dụng `vector` để lưu trữ dữ liệu
